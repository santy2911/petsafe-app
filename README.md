# 🐾 PetSafe - Sistema Multiplataforma para Gestión de Centros de Adopción y Recuperación de Mascotas

**PetSafe** es una plataforma digital integral diseñada para centralizar la gestión de refugios de animales y facilitar la adopción responsable y la recuperación de mascotas perdidas en España.

---

## ✨ Características Principales

- **Gestión completa de mascotas** (registro, edición y seguimiento)
- **Solicitudes de adopción** con flujo de aprobación
- **Reportes de mascotas perdidas** con geolocalización en tiempo real
- **Mapa interactivo** de mascotas perdidas
- **Sistema de notificaciones** (push y correo electrónico)
- **Gamificación** con puntos y recompensas para incentivar la participación
- **Panel de administración** para refugios
- **Interfaz responsive** y moderna

---

## 🛠️ Tecnologías Utilizadas

- **Frontend / App Móvil**: Flutter + Dart
- **Web**: Flutter Web
- **Backend**: Java + Spring Boot (API REST)
- **Base de Datos**: MySQL
- **Autenticación**: JWT
- **Despliegue Web**: Vercel
- **Control de versiones**: Git + GitHub

---

## 🚀 Cómo Ejecutar el Proyecto Localmente

### Requisitos
- Flutter SDK instalado
- Android Studio / Xcode (opcional para emuladores)
- MySQL (para el backend)

### Pasos

```bash
# 1. Clonar el repositorio
git clone https://github.com/santy2911/petsafe-app.git
cd petsafe-app

# 2. Instalar dependencias
flutter pub get

# 3. Ejecutar la aplicación Web
flutter run -d web

# 4. Ejecutar en emulador Android/iOS
flutter run
```
